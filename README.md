# fortee Talk Vote Support

fortee のスタッフ投票ページで、メモ入力とプロポーザルのフィルタリングを補助する Chrome 拡張機能です。

この拡張は非公式ツールです。このツールに関する問い合わせは、こうの（ https://x.com/hk_it7 ）までお願いします。

また、公式のアップデートにより使用できなくなる可能性があります。

本ツールの利用によって生じたいかなる損害についても、作者は責任を負いません。

## 機能

### 📝 インラインメモ

各プロポーザルカードの下部にメモ入力欄を自動で追加します。

- Vue.js の動的レンダリング完了後に自動注入
- 入力後 1 秒で自動保存（Chrome ローカルストレージ）
- メモあり → 黄色ボーダーで視覚的に区別
- キー：プロポーザルのタイトルテキスト

### 🔍 フィルタリング

拡張機能のポップアップからプロポーザルの表示/非表示を切り替えます。

**投票状態フィルタ**
- 全て
- 投票済み（`.btn.btn-primary` が存在するもの）
- 未投票（`.btn.btn-primary` がないもの）

**スコアフィルタ**（複数選択可）
- +2 / +1 / 0 / -1 / -2

### 💾 エクスポート

ポップアップの Export ボタンからメモをエクスポートできます。

- **JSON**: タイトルとメモのマッピングをまるごと出力
- **CSV**: `Title,Memo` の 2 カラム形式

## インストール

1. このリポジトリをクローンまたは ZIP でダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の **デベロッパーモード** を ON にする
4. **パッケージ化されていない拡張機能を読み込む** をクリック
5. このフォルダを選択

拡張機能を更新したあとは **再読み込みボタン** を押してください。
（content script の更新はページのリロードも必要です）

## 使い方

### メモを書く

Fortee のスタッフ投票ページを開くと、各プロポーザルカードの下部にテキストエリアが表示されます。入力した内容は自動保存されます。

### フィルタリング

拡張機能アイコンをクリックしてポップアップを開き、フィルタボタンを押すとページの表示が切り替わります。

「全て」に戻すと非表示にしたカードが再表示されます。

### エクスポート

ポップアップの **Export JSON** または **Export CSV** ボタンを押すとダウンロードが始まります。

## ストレージ

Chrome のローカルストレージに保存されます（サーバーへの送信なし）。

```json
{
  "memos": {
    "プロポーザルのタイトル": "メモの内容",
    ...
  }
}
```

## ファイル構成

```
fortee-talk-vote-support/
├── manifest.json     # 拡張機能の設定
├── popup.html        # ポップアップ UI
├── popup.js          # ポップアップのロジック
├── popup.css         # ポップアップのスタイル
├── content.js        # ページへの注入スクリプト
├── content.css       # 注入するスタイル
├── background.js     # Service Worker
├── icons/            # アイコン画像
```

## 動作環境

- Chrome / Chromium 系ブラウザ
- 対象ページ: `https://fortee.jp/*`
- Manifest V3
